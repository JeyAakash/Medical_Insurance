# Medical Insurance Cost Prediction using Multiple Linear Regression

## 📌 Project Overview
This project predicts medical insurance charges using the Multiple Linear Regression algorithm. It uses demographic and health-related information such as age, BMI, smoking status, and number of children to estimate insurance costs.

---

## 📂 Dataset
The dataset contains the following features:

- Age
- Sex
- BMI
- Children
- Smoker
- Region

**Target Variable:**
- Charges (Medical Insurance Cost)

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## 📈 Machine Learning Algorithm

- Multiple Linear Regression

---

## ⚙️ Project Workflow

1. Load the dataset
2. Encode categorical features
3. Split the dataset into training and testing sets
4. Train the Multiple Linear Regression model
5. Predict insurance charges
6. Evaluate the model using R² Score and MSE
7. Visualize Actual vs Predicted values

---

## 📊 Model Performance

- **R² Score:** 0.7833
- **Mean Squared Error (MSE):** 33635210.43
